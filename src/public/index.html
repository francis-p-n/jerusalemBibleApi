API Endpoints

Every API request returns standard JSON format including status codes, execution duration, and timestamp details.

GET /api/v1/bible/books List all 73 Catholic books of the Bible in order
GET /api/v1/bible/books/:slug Retrieve metadata & list of chapters of a book
GET /api/v1/bible/passage?ref=:query Fetch specific passages (e.g. John 3:16-18, Wisdom 3)
GET /api/v1/bible/search?q=:keyword Search keywords across scriptures with full pagination

Developer Snippets

JavaScript Fetch Example 
// Fetching a Scripture passage in Javascript
const fetchPassage = async (reference) => {
  const url = `http://localhost:3000/api/v1/bible/passage?ref=${encodeURIComponent(reference)}`;
  try {
    const res = await fetch(url);
    const json = await res.json();
    if (json.success) {
      console.log(`Reference: ${json.data.reference}`);
      json.data.verses.forEach(v => {
        console.log(`[Ch ${v.chapter}:${v.number}] ${v.text}`);
      });
    }
  } catch (err) {
    console.error("API request failed:", err);
  }
};

fetchPassage("John 3:16-18");
